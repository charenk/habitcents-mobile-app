# Design context (mirrored)

These files are mirrored copies of the canonical planning docs, which live in the
umbrella folder OUTSIDE this repo (`HabitCents/docs/`, not version-controlled on
GitHub). They are mirrored here so a GitHub-linked design session (Claude design)
has the full context in one repo.

- Mirrored: 2026-07-03. If in doubt, the umbrella copies win.
- Path mapping: references like `docs/leak-scan-spec.md` in these files resolve to
  `docs/design-context/leak-scan-spec.md` in this repo. References to
  `mobile-app/...` resolve to the repo root (e.g. `constants/strings.ts`,
  `constants/theme.ts`, `types/habit.ts` for the 7 micro-lessons).
- Not mirrored: `habitcents-plan-v2.html` (large interactive report; the status
  banner content is summarized in `phase-2-next-steps.md` and `CLAUDE.md`).
